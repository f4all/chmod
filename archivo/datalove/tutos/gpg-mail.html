Set up GPG-enabled email client with TOR on OSX.

 

Set up GPG-enabled email client on OSX.

1- Download and install thunderbird. http://www.mozilla.org/en-US/thunderbird/all.html (select language and OS)

2- Download and install GPGTools. https://github.com/downloads/GPGTools/GPGTools/GPGTools-20120318.dmg

3- Open GPG keychain access:
a- create new key.
b- add proper information. (name and email address)
c- click Generate key.
d- create secure password and confirm.
e- close Open GPG keychain access.

4- Open Thunderbird:
a- Configure your email account when asked.
b- Go to menu > tools > add ons.
c- search for Enigmail, install it.
d- Go to menu > tools > account settings.
e- Select the email account you setup on the beginning.
f- Select OpenPGP security and activate OpenPGP support (Enigmail).
g- Select second option (use specific OpenPGP ID key)
h- Click "Select keyâ€¦", select the key generated on step 3c and accept.
i- On the default options for message redaction activate all options.

5- Open GPG keychain access:
a- Select your previously generated key.
b- Right click it and export it with your desired name.
c- Share your public key with other trusted parties.
d- Send encrypted e-mail to trusted parties in possession of your public key.

6- BE SAFE ALWAYS SEND YOUR EMAILS WITH TOR ENABLED.
a- in Thunderbird go to menu > tools > add ons.
b- search for TorBirdy, install it.
d- download and install Tor Browser Bundle. https://www.torproject.org/download/download-easy.html.en
e- run Tor Browser bundle, you will not use it, but you will use TOR which is enabled with Vidalia automatically
when Tor browser bundle starts.


(JUST WAIT UNTIL YOU SEE ON TOR BROWSER BUNDLE : Congratulations. Your browser is configured to use Tor. THIS MUST APEAR ON THIS URL: https://check.torproject.org.)


f- Activate TorBirdy on Thunderbird, this will force all connections made by thunderbird to go through the TOR network.
g- Send your encrypted e-mails securely through the TOR network.
