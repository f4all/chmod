IRC Networks - Smaller Networks

The smaller IRC networks - less than 1,000 maximum users

For convenience and speed, the (many) small networks are listed on two pages. This page contains the more established and larger ones, whereas page two lists the newer, smaller and less stable small networks - and those which didn’t fit on this page. This doesn’t mean that they’re bad - many of them are worth a try.

Index of server lists

BeyondIRC
Originally formed by several NewNet servers, it still retains a degree of similarity.
Servers: 9
Locations: Norway, USA
Default port: 6667
Max users: ~600
Services: Channel (ChanServ), Nick (NickServ), Memo (NoteServ)
Started: September 1996
Game channels: #!gametrivia, #!gamehangman, #!game_tictactoe,

!game_blackjack

Operator channel: #Idleland
Server list.

StarLink
[Note: Don’t confuse this network with Starlink-IRC below, which split away. Starlink itself is in transition, dealing with domain name changes, trying to rebuild its user base, and changing services etc. This entry will continue to be amended or moved as things stabilize.]
Servers: <10
Locations: Australia, UK, USA, Sweden
Default port: 6667
Max users: ~200 for now [was >2000 prior to the current problems]
Services: Channel (ChanServ)
Started: July 1996
Game channels: Quite a few.
Administrative channel: #Oasis
Restrictive policies: No warez, sex, porn channels
Server list.

StarLink-IRC
Not to be confused with StarLink, which is older and larger.
Servers: 7
Locations: Canada, Netherlands, Sweden, USA
Default port: 6667
Max users: est. 350
Services: Channel
Started: January 1997
Operator channel: #terranova
Restrictive policies: No warez/sex/porn channels
Server list

XWorld
An extremely friendly and well-run network. Can’t but recommend it.
Servers: 16
Locations: Brazil, Canada, Germany, UK, USA
Default port: 6667
Max users: 400+
Services: Channel (XS), Profile (Proserv), Game (Gameserv)
Started: September 1996
Game channels: #trivia, #hangman, #acronym, #bj
Operator channel: #xworld
Server list.

MultiNET (LiquiNet/UpperNet)
Very relaxed atmosphere, this product of the LiquiNet/UpperNet merger is an excellent place to escape to with your friends for peaceful meetings under good conditions if you’re not willing to bother with formalities like registration and stuff.
Servers: 10
Locations: Bulgaria, Czech Rep., Netherlands, Sweden, USA
Max users: <100
Services: None
Operator channel: #oper
Server list

SorceryNet
An ambitious project to create a new network which will be “the best network on IRC”, running on guidelines similar to DALnet, managed by experienced admins of former DALnet servers.
Servers: 6
Locations: Sweden, USA
Default port: 9000
Max users: >200
Services: Nick (NickServ), Channel (ChanServ), Memo (MemoServ), News/Info (InfoServ)
Started: January 1997
Operator channel: #SorceryNet
Server list

SuperChat
This describes itself as a strict family network. Y’know, the sort you’d let your kids onto etc… Quite friendly and well-run. I dare say you could let your kids onto it.
Servers: 8
Locations: Australia, Canada, USA
Default port: 6667
Max users: >450
Services: Channel. located in #cservice
Alternative access: Java, Telnet
Started: Early 1996
Operator channel: #superchat
Restrictive policies: No warez, sex, hacking etc. channels
Server list

Valhall.Net
A tribute to Nordic mythology and Atari computers. A well-run network managed by experts. Their operations now include Linux and QNX.
Servers: 11
Locations: Germany, Sweden, USA
Default port: 7000
Max users: 100+
Services: Channel (ChanServ), Nickname (NickServ), Memo (MemoServ), Help (HelpServ)
Started: May 1997
Operator channel: #valhall
Server list

WarpedNet
Part of what used to be the Innernet now goes by this name as a network promising to let users run their channels themselves and keep the operators where they should be, in the background, running the servers. The #Amiga channel appears to be most prominent.
Servers: 13
Locations: Australia, Canada, UK, USA
Default port: 6667
Max users: est. 150+
Services: None
Started: October 1996
Operator channel: #speire
Server list.

World Wide Free-Net IRC network (WWFIN)
Strictly organized network run by freenets all over, this should appeal to everyone seeking a low-noise network and interested in the freenet concept.
Servers: 9
Locations: Canada, Germany, USA
Max users: 200+
Services: Channel (Guardian)
Started: before October 1995
Game channels: #chaos
Operator channel: #irc-admin
Restrictive policies: Total ban on warez, sex, drugs, ads, racism.
Server list.
